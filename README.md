# WhosThatColleague

A simple game that helps you to learn the names of your colleagues (if you happen to work at @wearespindle or Voys)

## Getting started

```bash
$ git clone git@github.com:confususs/whosThatColleague.git 
$ cd whosThatColleague
$ npm ci
$ npm start
```
